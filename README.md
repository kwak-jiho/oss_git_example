### Week1-2 오픈소스소프트웨어 개요
## 1. OSS ##
* 소프트웨어 저작권 소유자가 모든 사람에게 소스코드를 게시, 사용, 복사, 수정 및 배포할 권리를 부여한 소프트웨어
## 2. OSS License ##
* 오픈소스 소프트웨어의 사용, 복제, 수정, 배포 권한의 범위를 지정

- free SW vs OSS 차이점
